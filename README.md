## 📖 About

**Bible Undercover** is a browser-based multiplayer party game where players receive secret Biblical words. Most players receive the same word, while one or more **Imposters** receive a closely related word. Through discussion and voting, players must identify the imposter before time runs out.

Perfect for:

* ⛪ Churches
* 👨‍👩‍👧‍👦 Youth Fellowships
* 📖 Bible Study Groups
* 🎉 Christian Gatherings
* 🏕️ Camps & Retreats

---

## ✨ Features

* 👥 3–20 Players
* 😈 One or Multiple Imposters
* 📖 Biblical Word Pairs
* 🎯 Easy, Medium & Hard Difficulty
* 🗂️ Multiple Categories
* ✍️ Custom Player Names
* 👁️ Press & Hold Secret Reveal
* ⏱️ Configurable Discussion Timer
* 🗳️ Voting System
* 🏆 Round Scoreboard
* 🎉 Victory Celebration
* 📱 Responsive Design
* 🌙 Modern Dark Theme
